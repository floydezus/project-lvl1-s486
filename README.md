
Hexlet project lvl 1 "Brain games".

# Installation

    npm i brain-games-flz

# Commands
 
   Even number or not:    
        brain-even 

   Calculate the result for two numbers:
        brain-calc

    The greatest common divisor for two numbers:
        brain-gcd

    What element of arithmetic progression is missing:
        brain-progression

    Is the number prime:
        brain-prime
